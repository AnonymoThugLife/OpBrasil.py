Você precisaria do Python 2.7+. Você pode obtê-lo nos links na internet como sua primeira consulta do Google.

Os módulos que você precisa são BeautifulSoup, bs4, requests, socks, socks5 ... para instalar --> No seu terminal:

pip install BeautifulSoup

pip install bs4

caso contrário, você receberá a mensagem de erro "você precisa instalar: bs4" (Precisa do -> pip install bs4 )

O próximo módulo que você precisa é:

pip install requests

caso contrário, você receberá a mensagem de erro "você precisa instalar: requests"

último mas não menos importante, o último módulo:

pip install socks

caso contrário, você receberá a mensagem de erro "socks"

Se você tiver todos os módulos, tente reiniciar a ferramenta .. comando "python OpBrasil.py" se você não receber nenhum erro, ele será executado e você estará pronto para colocar a merda offline <3
